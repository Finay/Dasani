DASANI
---
DAta Science with ANIme. A quick project to practice data science skills and building dashboards.

## Setup
```bash
# Set up a virtual environment
$ python -m venv env

# Install requirements
$ pip install -r requirements.txt
```

## Usage
```bash
# Start dashboard
$ python main.py
```
